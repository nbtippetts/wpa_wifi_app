The Web interface

Home Page


Fields:

  • Name:indicates the name of the grow plan you have selected in the Nutrient table page

  • Starting date:The starting day of the grow plan and how many days have passed

  • Status: indicate the status

  • Average temperature 24h: indicates the average temperature during the last 24 hours

  • Today light: indicate the light quantity of the current solar day (Lumen per day)



Nutrient Table



Fields:

  • Name:The name of the grow plan can be selected

  • Week:The weeks from the beginning of the plan

  • Period weeks: the delta period in weeks, cont how many weeks the below quantity are applied

  • Phase: Indication of the plant phase (optional)

  • Nutrient(ml): the quantity of fertilized in ml that will be dropped every week in the irrigation water reservoir (4 different nutrient can be selected)

  • Minimum Light: the quantity of daily light required for the plant. If this quantity is not reached during the day then the light will be switched ON

  • Maximum Light ON (minutes/day): The maximum number of minutes the light can be switched ON during one day

  • Water Pump Cycle (minutes): The time in minutes the between two pump activations. The first pump activation is calculated as the number of minutes after 00:00. Pump cycle will be reset at the end of each day. (example: if 420 minutes are set, meaning 7 hours, then the pump is activated at 7:00am, 14:00 and 21:00 for each day). If PumpCycle value is set higher that one day (>60*24=1440) then the pump will be activated skipping the number of days as result of PumpCycle value divided by 60*24. Example, if PumpCycle=3000 then int(3000/1440) = 2, the pump will be activated every 3 days inside the week (week day 1 and week day 4)

  • Water pump ON (seconds): the number of seconds the pump is ON.


Buttons:

  • Remove: Remove the current table

  • Modify: Modify the field

  • Select: Select the current table for as growing plan starting from current day



Setting





Calibration:

This section is used to calibrate the doser quantity. This operation should be done before starting the grow plan.



Light Setting:

Set the time after the light is switched ON if the light until that time does not reach the minimum set in the growing plan
